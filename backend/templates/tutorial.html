How to Use AccessBank

Getting Started with Chat

1

Start a Chat

Click the "Start Chat" button on the home page to begin your conversation with AccessBank AI.

2

Type Your Message

Type your banking-related question in the chat input field and press Enter or click Send.

3

Voice Commands

Click the microphone button to use voice input. Speak clearly and wait for the response.

Accessibility Features

1

High Contrast Mode

Enable high contrast mode for better visibility. Click the Accessibility icon and toggle "High Contrast Mode".

2

Screen Reader Support

Our interface is optimized for screen readers. Enable screen reader optimization in the Accessibility settings.

3

Text Size Adjustment

Adjust text size using the slider in Accessibility settings. Choose between 12px and 24px.

4

Voice Commands

Use voice commands for navigation and actions. Enable voice commands in Accessibility settings.

Available commands:

  • "Start chat" - Open the chat interface
  • "Enable high contrast" - Toggle high contrast mode
  • "Enable screen reader" - Toggle screen reader optimization
  • "Increase text size" - Make text larger
  • "Decrease text size" - Make text smaller

Language Support

1

Select Language

Choose your preferred language from the dropdown menu in the chat interface.

2

Translate Chat

Click the translate button to convert the conversation to your selected language.
